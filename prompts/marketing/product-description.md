# Product Description

## Purpose
Generate a persuasive product description that highlights features, benefits, and competitive advantages.

## Best For
- ecommerce copy
- landing pages
- product marketing

## Recommended Model
ChatGPT / Claude / Gemini

## Inputs
- product name
- product type
- target audience
- key features
- benefits
- differentiators
- word count

## Prompt
Write a compelling 150-word product description for a new eco-friendly water bottle.

Requirements:
- Highlight its features
- Explain its benefits
- Show why it is better than competitors
- Keep it persuasive, clear, and customer-focused
- Make the description sound polished and market-ready

## Example Input
- Product: eco-friendly water bottle
- Word count: 150
- Audience: environmentally conscious consumers

## Example Output
A concise product description covering materials, usability, sustainability benefits, and competitive edge.

## Notes
- Works well for ecommerce and catalog copy
- Better results come from adding exact features, materials, and audience details
