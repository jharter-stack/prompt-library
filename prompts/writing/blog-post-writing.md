# Blog Post Writing

## Purpose
Generate a structured blog post on a defined topic with a clear intro, main points, supporting detail, and conclusion.

## Best For
- content marketing
- blog drafting
- educational articles

## Recommended Model
ChatGPT / Claude / Gemini

## Inputs
- topic
- target word count
- audience
- tone
- structure requirements

## Prompt
Write a [WORD COUNT]-word blog post on [TOPIC].

Requirements:
- Include an introduction
- Include [NUMBER] points with supporting evidence
- Include a conclusion
- Keep the writing clear, engaging, and informative
- Make it suitable for a general audience

## Example Input
- Topic: meditation for stress relief
- Word count: [WORD COUNT]
- Audience: [AUDIENCE]
- Tone: [TONE]

## Example Output
A blog post with:
- intro explaining stress and meditation
- 3 evidence-based benefits
- conclusion summarizing why meditation helps

## Notes
- Good starter prompt for blog content
- Can be expanded by adding audience, tone, SEO keywords, or CTA requirements
